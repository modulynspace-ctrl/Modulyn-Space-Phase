# Modulyn Space – Phase 1: Premium UI Foundation

You are an award-winning creative director, senior UI/UX designer, and senior frontend architect.

Do NOT generate a generic interior design website.

Create a luxury digital experience that feels comparable to premium architecture studios and modern luxury brands.

The feeling should be:

* Apple
* Lexus
* Architecture Magazine
* Minimal Luxury
* Editorial Design

The website must immediately communicate:

"These people build premium interiors with transparency, craftsmanship, and timeless quality."

---

## Technology

Create the project using:

* React
* Vite
* TypeScript
* Tailwind CSS

Use clean architecture with reusable components.

Do NOT create backend functionality.

Do NOT connect any database.

Do NOT create authentication.

Use placeholder data where required.
